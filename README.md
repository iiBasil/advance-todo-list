# Advanced To-Do List

A simple task management web application built using HTML, CSS, and JavaScript.

## Overview

This project allows users to manage daily tasks in an organized way.  
It focuses on DOM manipulation, user interaction, and storing data locally.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Task filtering (All / Completed / Active) *(إذا موجودة عندك)*
- Data saved using LocalStorage
- Responsive design

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project files
2. Open index.html in your browser
or
3. Use VS Code Live Server

## Purpose

This project was built to practice JavaScript logic, event handling, and state management in a simple task management system.
